# Changelog

## 0.1.0-free - Starter free release

Initial free/starter release of QuietGate.

Private homelab access with Tailscale + Traefik, without opening public ports.

Included features:

- Traefik reverse proxy
- Homepage dashboard
- Uptime Kuma private monitoring
- Whoami routing test app
- Tailscale host-based private access model
- `nip.io` quick-start hostnames
- healthcheck script
- exposure-check script
- quick-start guide
- architecture guide
- Tailscale guide
- Traefik guide
- DNS options guide
- adding apps guide
- security checklist
- troubleshooting guide
- support boundaries

Intentional exclusions:

- no public port forwarding
- no ARR stack
- no torrent clients
- no VPN containers
- no Cloudflare Tunnel
- no Kubernetes
- no Authentik/SSO
- no Jellyfin in the free starter release
